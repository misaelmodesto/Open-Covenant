# Open-Covenant ⛪

**Open-Covenant** é um ecossistema open-source de gestão eclesiástica. Nosso objetivo é fornecer ferramentas robustas para igrejas, focando em transparência financeira, educação bíblica e automação de processos.

## 🛠 Tech Stack (Planejada)
- **Back-end:** Go (Golang) - *Foco em performance e concorrência.*
- **Front-end:** Next.js (React + TypeScript) - *Interface moderna e responsiva.*
- **Banco de Dados:** PostgreSQL - *Integridade de dados e consultas complexas.*
- **Integrações:** n8n / Node-RED para Bot de WhatsApp e IA.

## 📋 Módulos do Projeto
- [ ] **Financeiro:** Gestão de dízimos, ofertas, tesouraria e patrimônio.
- [ ] **Social:** Gestão de cestas básicas e assistência.
- [ ] **Ensino:** Escolas bíblicas, turmas e discipulado.
- [ ] **Documentação:** Gestão de atas e ofícios com auxílio de IA.

## 🤝 Como Contribuir
Este projeto nasceu para ser comunitário. Desenvolvedores, designers e líderes são bem-vindos para colaborar.
